開始測驗
「我怎麼可能成為無家者！？」

有人說出來社會走跳，是三分靠作弊、七分靠後壁。

但是人生就是這樣，衰起來的時候連穿新鞋出門都會踩到黃金。

假如你有天衰到後壁只剩鳳梨、香蕉、芭辣，你會是艋舺走撞裡哪個無家者呢？

繼續

你回頭望去，車子的停車格號碼是？

A

我不知道

B

108

C

87

Q2

剛剛老闆找你去他的小房間，稱讚你工作認真並且表現優秀，

今年要給你二十萬績效獎金，下個月一號入帳。

興奮的你拿到這筆錢以後打算怎麼處理它呢？

A

二十萬全部存定存

B

買手機犒賞自己。

剩下的一時之間還沒想到要幹嘛，之後再說啦！

C

喔耶太爽了！

花了十五萬請全公司吃大餐。

D

解約一筆五十萬定存加上這筆二十萬，全部投資在朋友介紹的尚未上市股票上。

Q3

最近工作壓力好大，客戶的要求越來越多

老闆的標準也越來越高，連續加班好幾天了

感覺實在是撐不太下去了

我該如何是好？

A

老闆和客戶都這麼機車，

老子不幹啦！

B

不知道怎麼辦，

還是先打場傳說對決好了。

C

先釐清一下客戶和老闆的想法，再看看能夠怎麼去解決現在的問題吧！

Q4

深夜跟家人大吵一架後，你負氣走出家門，沒有任何目標的在街上遊走。

夜深了，你走到台北車站，看到有人鋪了紙箱睡袋準備就寢，

你也找了個空位坐了下來，迷迷糊糊開始打起瞌睡。

朦朧之中有個小聲音在你耳邊說：

「欸，新手對吧？第一天在外面過夜喔！你覺得啊，在街頭上是男生還是女生比較容易生存下來啊？」

你睜開雙眼看見一個半透明的小仙子在你眼前飛舞。

「呵呵呵你醒了欸！這樣好了，給你菜鳥特別優惠，今天晚上你可以選擇性別進行第一次街頭體驗，

請問你想要當男生還是當女生呢？

我選男生！

當然是女生！

Q5

最終還是被老闆炒魷魚了，也不好意思回去找家人幫忙，

你在去吃飯的路上巧遇國中同學阿茂，

對方問起你現在在做什麼，

要怎麼回應呢？

A

「什麼事情都要向你報備就是了」

你頭也不回的離開了阿茂。

B

丟了工作實在不知道該怎麼說起才好，你愣了一下，久久不知道該如何說起。

C

「最近過的不是很好啊，...」

你和阿茂聊起了自己的生活狀況。

Q6

能借的錢都借了，能找的幫忙也都找過了，

處處碰壁，失去了工作、繳不出房租而即將被房東趕出來的你，

此時茫然地躺在床上，世界很大，但你卻不知道有那裡可以去，

連明天要怎麼活下去都不知道。

此時的你看向門邊上次烤肉留下來的木炭...

A

把房間門窗關起，門縫塞住，

點好木炭後躺回床上，閉上眼睛，

等一下就沒有煩惱了。

B

「活著好痛苦，可是死也好可怕 ！」

人生好難，到底要怎麼辦呢？

C

把房間門窗打開，點好木炭後開始烤起秋刀魚，管它的，先吃飽再說吧。

Q7

在房間裡燒木炭的事情被鄰居發現，於是被房東趕出來了。

你不知道要去哪裡，「乾脆去流浪好了！」

你正好走到岔路口，左轉是沒有人的河濱橋下，右轉是熱鬧的市區。

A

左轉去河濱橋下涵洞躲起來，

離群索居，過著吃野菜、補河魚，

燒柴火的游牧生活。

B

晚上睡在無人的河濱橋下，

白天到市區找食物與工作，

過著通勤的生活。

C

右轉到熱鬧的市區，

說不定人多的地方就有活下去的機會。

Q8
